# 내 프로필 (profile.md) — AI가 나에게 맞추는 파일

> 신청폼에 적은 내용을 그대로 옮긴다. AI는 매 세션 이 파일을 읽고
> 내 눈높이(코딩 수준·글 표현)에 맞춰 설명하고, 기획 인터뷰(blueprint)에서
> 여기 적힌 내용을 선답변으로 삼아 "맞습니까?" 확인부터 시작한다.
> 안 채워도 동작한다 — 채울수록 AI가 나를 잘 안다.
>
> 주의: 실명·연락처·계좌 같은 민감 정보는 넣지 않는다 (이 파일은 깃허브에 올라간다).

## 만들고 싶은 것
건강검진 결과지를 받고 당황하지 않게, 수치를 입력하면 Part 1~10으로 내 건강을 쉽게 풀어주는 "평생건강가이드북". 의사 편지·건강 점수·믿는 구석·경고·용어사전·다음 검진까지 프로젝트·영양제 점검·건강습관·미래 위험·마무리 약속으로 구성.

## 누가 쓰나 (타겟)
40~70대 우리 가족 (본인 + 80세 어머니 + 20대 아들)
