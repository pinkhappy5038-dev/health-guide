# 내 프로젝트 기획서 (plan.md) — 평생건강가이드북

> 상세 지도(Part 1~10 전체 설계)는 `docs/plan/00-guidebook-map.md` 에 있다.
> progress 훅이 아래 체크박스를 읽어 진행상황 화면을 만든다.

## 한 문장 소개
나는 **40~70대 우리 가족** 을 위해 **건강검진 결과지를 쉽게 풀어주는 평생건강가이드북** 을, 핵심 기능 **수치 입력 → Part 1~10 책처럼 넘겨 보기** 로 만든다.

## 데이터 (뭘 저장하나)
- 표: **검진기록** (필드: 성별, 키, 몸무게, 혈압, 혈당, 콜레스테롤, 간수치, 신장수치, 빈혈 / 나중에: 출생연도, 직업, 다음 검진 예정, 생활습관 문진)
- 표: **습관체크** (필드: 날짜, 항목, 체크 여부 — Part 8용)
- 지금은 localStorage(브라우저 저장) → 가족이 각자 쓰게 되면 Supabase 로 이사
- 관계: 한 사용자가 여러 검진기록 → 1:N

## 조립 순서 (블럭 = 기능 하나 / 스텝 = 한 번에 시킬 일 하나)

### 블럭 0. 프로토타입 (완료 — `prototype/index.html`, 파일 1장짜리 앱)
- [x] 0-1. 검진 수치 입력 화면 + 정상/주의/위험 자동 판정 — 확인: 값 넣으면 배지가 바뀐다
- [x] 0-2. 책 뼈대: 차례 화면 + 한 파트씩 넘기기(◀이전/다음▶ 큰 버튼) — 확인: 브라우저에서 넘겨진다
- [x] 0-3. Part 2 자리에 '한눈에 보기' 배치 — 확인: 저장하면 Part 2 로 이동

### 블럭 1. 입력 화면 확장 — 0단계 장보기 (완료 기준: 새 빈칸들이 보이고 저장된다)
- [x] 1-1. 출생연도·직업·다음 검진 예정(연월) 칸 추가 — 확인: 입력 화면 '나에 대해' 묶음에 새 칸 3개 보이고, 저장하면 남고, Part 2 점수엔 안 나옴 (2026-07-11 검증 통과)
- [x] 1-2. 생활습관 문진 5개(운동·단 음료·흡연·음주·짜게) — 확인: '생활 습관' 묶음에 드롭다운 5개, 골라 저장하면 남고, Part 2 점수엔 안 나옴 (2026-07-11 검증 통과)

### 블럭 2. Part 2 내 건강 점수 (완료 기준: 영역별 점수 + 별점이 보인다)
- [x] 2-1. 영역별(혈관·혈당·간·콩팥·체중) 100점 점수 계산 — 규칙: 정상100·주의60·위험30 평균, 값 없는 항목 제외, 영역에 값 0개면 '아직 점수 없음' (2026-07-11 검증 통과)
- [x] 2-2. 황금 별점(★) 표시 — 확인: "간 건강 100점 ★★★★★" 형태로 Part 2 상단 카드에 보임 (2026-07-11 검증 통과)

### 블럭 3. Part 3 내 몸의 믿는 구석 (완료 기준: 정상 항목만 칭찬으로 보인다)
- [x] 3-1. 정상 판정 항목만 영역별로 모아 보여주기 + 초록 인사말 + 빈 경우 안내 — 확인: Part 3에 정상만 ✅로 뜨고 위험은 안 나옴, 정상 0개면 부드러운 안내 (2026-07-12 검증 통과). 차례에서 '준비 중' 딱지 뗌.

### 블럭 4. Part 4 내 몸에 보내는 경고 (완료 기준: 주의·위험만 생활 비유로, 신호등)
- [x] 4-1. 주의·위험 항목만 모아 생활 비유 문구 + 신호등(🔴위험 먼저·🟡주의) — 확인: 섞인 값이면 정렬돼 뜨고, 모두 정상이면 "경고할 게 없어요 🎉", 미입력이면 안내 (2026-07-14 검증 통과). 차례 '준비 중' 뗌.

### 블럭 5. Part 1 의사 선생님의 편지 (완료 기준: 수치에 맞는 편지가 자동으로 써진다)
- [x] 5-1. 규칙 기반 편지 조립 — ①인사 ②전체그림 ③구체적 칭찬(정상 영역 + 좋은 습관 답 연결: 술·단음료·운동·짜게) ④신경 쓸 것(위험 우선 최대 3개, 생활 비유) ⑤격려+다음 검진일. 조사(은/는·이/가) 자동 선택. 빈 경우 안내. (2026-07-14 검증 통과, 차례 딱지 뗌)
- 사용자 피드백 반영: "평소 습관" 같은 애매한 칭찬 금지 → 몸이 잘하는 일 구체적으로 + 문진 답이 좋을 때만 습관 연결 (지어내지 않기).

### 블럭 6. Part 5 그게 무슨 뜻이에요? (완료 기준: 용어를 쉬운말+내 수치로 보여준다)
- [x] 6-1. 결과지 용어 15개 쉬운말 사전 (생활 비유 한 줄) + 수치 입력 시 "📌 당신의 수치: ○○ [판정]" 개인화. 수치 없어도 사전은 보임. BMI 자동 계산 포함. (2026-07-14 검증 통과, 차례 딱지 뗌)
- 나중에: Pixar Style 일러스트 (글 먼저 원칙) / Part 1~4 글 속 용어 탭하면 설명 카드(인라인).

### 블럭 7. Part 10 의사의 마지막 이야기 (완료 기준: 철학+여정+약속 저장이 된다)
- [x] 7-1. 철학 한 마디(사용자 초안 그대로) + 내 여정 한 줄 자동 요약 + ✍️ 나와의 약속 한 줄 입력·저장·고치기 + "다음 검진에서 만나요 — ○○" — 확인: 약속 저장 후 새로고침에도 남음 (2026-07-14 검증 통과, 차례 딱지 뗌)
- 나중에: 다음 검진 수치 입력하는 날 옛 약속 다시 보여주기 (여러 검진 기록 기능 생길 때).

### 블럭 8. Part 8 오늘부터 하는 건강습관 (완료 기준: 매일 체크가 날짜별로 저장된다)
- [x] 8-1. 시작 습관 3개(물 한 컵·30분 걷기·야식 금지) 체크리스트 + 날짜별 저장(HABITS_STORE_KEY)
  + 🔥 연속 달성 표시 + 3/3 축하 문구 — 확인: 체크·새로고침 유지·연속달성·축하 모두 통과.
  버그 수정: 연달아 누르면 체크 덮어쓰던 문제(직전 상태 기준 계산으로). (2026-07-14 검증 통과, 차례 딱지 뗌)
- 나중에: 연령·성별 맞춤 습관 제안(AI 단계) / Part 6 미션·Part 7 영양제 시간표가 여기로 흘러들어옴 / 습관 추가·수정 UI.

### 블럭 9. Part 6 다음 검진까지 프로젝트 (완료 기준: 결승선+목표+미션이 자동으로 나온다)
- [x] 9-1. 🏁 결승선(다음 검진일) + 🎯 목표 수치(주의·위험 상위 3개, 지금→목표) + 📅 시즌 미션(문제 영역별 자동, 최대 4개) — 확인: 문제 섞인 값·전부 정상(유지 프로젝트)·미입력 3케이스 통과 (2026-07-14 검증, 차례 딱지 뗌)
- 나중에: 미션 체크 추적 + Part 8 체크리스트 연동 / 연령 맞춤·AI 제안(AI 단계) / 시즌 나누기(중간점검).

### 블럭 10. Part 9 앞으로 생길 가능성 높은 질환 (완료 기준: 갈림길이 수치·습관에서 자동으로)
- [x] 10-1. 규칙 기반 갈림길 — 수치+습관(단 음료·술·담배·짜게)으로 경로 발동(혈당·혈관·간·체중·흡연·콩팥),
  경로마다 1·5·10년 + 💚 "바꾸면" + 연령 한 줄(출생연도 있으면) + "예언 아님" 안내 + Part 6 연결 문구.
  확인: 경로 여러 개·위험 없음·미입력 3케이스 통과 (2026-07-14 검증, 차례 딱지 뗌)
