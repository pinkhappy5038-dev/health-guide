# 내 프로젝트 기획서 (plan.md) — 평생건강가이드북

> 상세 지도(Part 1~10 전체 설계)는 `docs/plan/00-guidebook-map.md` 에 있다.
> progress 훅이 아래 체크박스를 읽어 진행상황 화면을 만든다.

## 한 문장 소개
나는 **40~70대 우리 가족** 을 위해 **건강검진 결과지를 쉽게 풀어주는 평생건강가이드북** 을, 핵심 기능 **수치 입력 → Part 1~10 책처럼 넘겨 보기** 로 만든다.

## 데이터 (뭘 저장하나)
- 표: **검진기록** (필드: 성별, 키, 몸무게, 혈압, 혈당, 콜레스테롤, 간수치, 신장수치, 빈혈 / 나중에: 출생연도, 직업, 다음 검진 예정, 생활습관 문진)
- 표: **습관체크** (필드: 날짜, 항목, 체크 여부 — Part 8용)
- 지금은 localStorage(브라우저 저장) → 가족이 각자 쓰게 되면 Supabase 로 이사
- 관계: 한 사용자가 여러 검진기록 → 1:N

## 조립 순서 (블럭 = 기능 하나 / 스텝 = 한 번에 시킬 일 하나)

### 블럭 0. 프로토타입 (완료 — `prototype/index.html`, 파일 1장짜리 앱)
- [x] 0-1. 검진 수치 입력 화면 + 정상/주의/위험 자동 판정 — 확인: 값 넣으면 배지가 바뀐다
- [x] 0-2. 책 뼈대: 차례 화면 + 한 파트씩 넘기기(◀이전/다음▶ 큰 버튼) — 확인: 브라우저에서 넘겨진다
- [x] 0-3. Part 2 자리에 '한눈에 보기' 배치 — 확인: 저장하면 Part 2 로 이동

### 블럭 1. 입력 화면 확장 — 0단계 장보기 (완료 기준: 새 빈칸들이 보이고 저장된다)
- [ ] 1-1. 출생연도·직업·다음 검진 예정(연월) 칸 추가 — 확인: 입력 화면에 새 칸이 보인다
- [ ] 1-2. 생활습관 문진 4~5개(운동·단 음료·흡연·음주) — 확인: 선택하고 저장하면 새로고침에도 남는다

### 블럭 2. Part 2 내 건강 점수 (완료 기준: 영역별 점수 + 별점이 보인다)
- [ ] 2-1. 영역별(간·혈관·혈당 등) 100점 만점 점수 계산 — 확인: Part 2 상단에 점수 보인다
- [ ] 2-2. 황금 별점(★) 표시 — 확인: "간 건강 90점 ★★★★★" 형태로 보인다

### 블럭 3~. Part 3(믿는 구석) → 4(경고) → 1(편지) → 5(용어) → 10(마무리) → 8(습관) → 6(프로젝트) → AI 연결 → 7(영양제) → 사진인식
> 각 파트 상세는 `docs/plan/00-guidebook-map.md` 참조. 블럭은 만들 차례가 오면 스텝으로 쪼갠다.
